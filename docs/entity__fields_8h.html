Generated by entity_fields_generator.py. This file contains macros named ::ENT_REGISTER_FIELDS_0 through 10. These are intended to be used in the class definition of any user-defined EntityComponent. The macros declare the variables passed to it (with access modifier 'public') and override the functions EntityComponent::save() and EntityComponent::load() by automatically setting the fields or writing them to the binary save file. Any fields which do not require saving (e.g. pointers) should be omitted from declaration in these macros.
